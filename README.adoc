= aptupdatechecker

A system service for checking APT package and firmware updates, with desktop notifications.

== Overview

`aptupdatechecker` monitors the system for available updates and sends desktop notifications to alert users. It uses a privilege-separated architecture to ensure security and proper notification delivery.

The application checks for:

* APT package updates through apt update and cache inspection
* Firmware updates through fwupdmgr

The application sends notifications using the freedesktop notification system.

== Architecture

The service uses separate processes for different privilege levels:

* **System services** (run as root): Update the APT package cache
* **User services** (run as user): Check for updates and send desktop notifications

This separation ensures that notification code runs in the user session (with D-Bus access) while system operations run with appropriate privileges.

== Installation

Build the Debian package:

[source,bash]
----
dpkg-buildpackage -us -uc -b
----

Install the package:

[source,bash]
----
sudo apt install ../aptupdatechecker_0.1.0-1_amd64.deb
----

Enable the user timers for notifications:

[source,bash]
----
systemctl --user enable --now aptupdatechecker_apt_update_checker.timer
systemctl --user enable --now aptupdatechecker_firmware_update_checker.timer
----

The system timer for APT cache updates enables automatically.

== Commands

The `aptupdatechecker` binary supports several operation modes:

`aptupdatechecker update`::
Updates the APT cache (requires root, no notifications). Used by the system service.

`aptupdatechecker check`::
Checks for APT updates and sends notifications (user mode). Does not update the cache.

`aptupdatechecker fwupd`::
Checks for firmware updates and sends notifications (user mode).

`aptupdatechecker apt`::
Legacy mode: updates cache and checks for APT updates (requires root and D-Bus access).

`aptupdatechecker`::
Legacy mode: runs both APT and firmware checks.

== Systemd Services

=== System Services

Located in `/lib/systemd/system/`:

`aptupdatechecker_apt_cache_update.service` / `.timer`::
Updates the APT package cache every 6 hours as root.

`aptupdatechecker_autoclean.service` / `.timer`::
Runs `apt autoclean` weekly to remove obsolete package files.

`aptupdatechecker_autoremove.service` / `.timer`::
Runs `apt autoremove` weekly to remove unnecessary packages.

=== User Services

Located in `/usr/lib/systemd/user/`:

`aptupdatechecker_apt_update_checker.service` / `.timer`::
Checks for APT package updates hourly and sends desktop notifications.

`aptupdatechecker_firmware_update_checker.service` / `.timer`::
Checks for firmware updates hourly and sends desktop notifications.

== Development

=== Building

Requirements:

* Rust toolchain (rustc >= 1.70)
* Debian build tools (debhelper, dpkg-dev)
* Development libraries: libapt-pkg-dev, libdbus-1-dev, pkg-config

Build the release binary:

[source,bash]
----
cargo build --release
----

Run tests:

[source,bash]
----
cargo test
----

Build the Debian package:

[source,bash]
----
dpkg-buildpackage -us -uc -b
----

=== Testing Manually

Test the APT cache update (requires root):

[source,bash]
----
sudo ./target/release/aptupdatechecker update
----

Test the APT update check (runs as user):

[source,bash]
----
./target/release/aptupdatechecker check
----

Test the firmware check (runs as user):

[source,bash]
----
./target/release/aptupdatechecker fwupd
----
